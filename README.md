# C# Console Guessing Game (Trò Chơi Đoán Số)

Đây là một ứng dụng Console đơn giản được xây dựng bằng C# (.NET 8.0) để thể hiện khả năng xử lý logic, vòng lặp, và xử lý nhập liệu (Input Validation).

### Kỹ năng đã thể hiện
* Sử dụng lớp `Random` để tạo số ngẫu nhiên.
* Logic vòng lặp `while` và câu lệnh điều kiện `if/else if/else`.
* **Xử lý lỗi nhập liệu mạnh mẽ** bằng `int.TryParse` và kiểm tra phạm vi.
* Hỗ trợ ký tự Unicode (tiếng Việt có dấu) trong Console.

### Cách Chạy
1.  Mở Solution file (`.sln`) trong Visual Studio 2022.
2.  Nhấn F5 để khởi động ứng dụng Console.
